# Rekap Plugin

rekap formatter

## Installation

### Requirements
- PHP >=5.5.0
- composer, <https://getcomposer.org/>

## Installation

Add this composer.json file

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:harryosmar/plugin-rekap.git"
        }
    ],
    "require": {
        "harryosmar/plugin-rekap": "^1.0"
    }
}
```

Then running
```bash
$ composer install
```
